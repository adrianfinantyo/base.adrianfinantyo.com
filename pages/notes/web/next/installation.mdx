# Next JS Installation Guide

## Prerequisites

- [Node.js](https://nodejs.org/en/download/)
- [Yarn](https://classic.yarnpkg.com/en/docs/install/#windows-stable)
- [Git](https://git-scm.com/downloads)

## Installation

1. Open a terminal and navigate to the folder where you want to create the project.

2. Run the following command to create a new project:

   JavaScript templates

   ```bash copy
   yarn create next-app
   ```

   TypeScript templates

   ```bash copy
   yarn create next-app --typescript
   ```

3. Navigate to the newly created project folder:

   ```bash copy
   cd my-app
   ```

4. Run the following command to install the required packages:

   ```bash copy
   yarn
   ```

5. Run the following command to start the project:

   ```bash copy
   yarn start
   ```

6. Open a browser and navigate to [http://localhost:3000](http://localhost:3000/) to see the project running.
